# ShyamEdge AI — Website (Next.js)

## Preview online (no install)
1) Create a new GitHub repo.
2) Upload **all files and folders inside this project** (make sure `package.json` is visible at the top level).
3) Go to Vercel → Add New → Project → Import your repo → Deploy.

After deploy, open:
- Website: `https://YOUR-URL.vercel.app`
- Admin placeholder: `https://YOUR-URL.vercel.app/admin`

## (Optional) Environment variables on Vercel
Settings → Environment Variables:
- `NEXT_PUBLIC_SITE_URL` = your Vercel URL
- `LEADS_TO_EMAIL` = shyamcreations42@gmail.com
